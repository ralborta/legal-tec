# Logos para la aplicación

## Archivos necesarios

Para que los logos se muestren correctamente, coloca estos archivos en la carpeta `ui/public/`:

### 1. Logo de WNS & Asociados
- **Archivo**: `wns-logo.png`
- **Ubicación**: `ui/public/wns-logo.png`
- **Tamaño recomendado**: Altura de ~28px (el código lo escala automáticamente)
- **Formato**: PNG con fondo transparente (preferible)

### 2. Logo de IA Solutions
- **Archivo**: `ia-solutions-logo.png`
- **Ubicación**: `ui/public/ia-solutions-logo.png`
- **Tamaño recomendado**: Altura de ~20px (el código lo escala automáticamente)
- **Formato**: PNG con fondo transparente (preferible)

## Ubicación de los logos en la UI

- **Logo WNS**: Se muestra en el Topbar (header), a la izquierda, antes del logo "IA Centro de Gestión"
- **Logo IA Solutions**: Se muestra en el Footer (parte inferior), junto al texto "Powered by"

## Notas

- Si los logos no se encuentran, la aplicación seguirá funcionando (se ocultan automáticamente)
- Los logos se escalan automáticamente manteniendo su proporción
- Se recomienda usar formatos PNG con fondo transparente para mejor integración visual
